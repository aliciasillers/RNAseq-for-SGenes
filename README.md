# RNAseq for Expression Analysis of Candidate Susceptibility Genes

Scripts are not generalized and will need to be modified to run with different input data. The order of the scripts is as follows:

1. getdata.sh
2. fastq.sh 
3. fastqc.sh
4. fastqcunzip.sh
5. starindex.sh
6. starmap.sh
7. convert.sh
8. count.sh
